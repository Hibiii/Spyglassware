# Spyglassware

This mod aims to bring some improvements to the vanilla spyglass in order to make it a viable alternative to OptiFine zoom and similar mods.

## Features Todo

- Disable zoom in animation
- Configurable zoom level
- Keybind to zoom with spyglass
- Mouse settings and cinematic camera toggle
- Overlay toggle
- Mute toggle
- Serverside overrides
